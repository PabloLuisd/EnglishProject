📘 English Project

Este projeto foi desenvolvido como parte da disciplina de programação para dispositivos moveis do curso de TADS - IFRS, com o objetivo de incentivar o aprendizado de inglês para crianças.

O trabalho consiste em imagens interativas, que retornam seus nomes em audio no inglês.

✨ Principais recursos:
● 🧠 Aprendizado por associação (imagem + palavra + som)
● 🔊 Áudio com pronúncia correta em inglês
● 🎨 Interface colorida e amigável para crianças
● 📚 Diversas categorias (animais, alimentos, objetos e muito mais)
